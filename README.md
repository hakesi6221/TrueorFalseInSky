# ⭕❌TrueorFalseInSky

2Dの〇×クイズゲームです。
クリックやタップでどちらか選び、できるだけ多くのクイズに成功することが目的です。

※このリポジトリは学習/ポートフォリオ公開を目的としており、一部の外部アセットはライセンス上の理由により除外しております。
